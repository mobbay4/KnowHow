# How to clone a repository from code.siemens

1. Make sure you have a local installation of Git on your computer
2. Go to the repository website and copy the SSH git clone link to your clipboard
![SSH clone link](./artwork/codeSiemens_clone_link.png)
3. Now, open a Git bash and move to the dictionary where you want to place the repository
4. Finally, type the following command while inserting the proper link and enter your SSH-key password if asked.

```shell
git clone "the copied link"
```

You should see something like this:
![Git Bash](./artwork/codeSiemens_git_bash.png)

Now, you have cloned the repository to the chosen folder!

[Git]: https://git-scm.com/
