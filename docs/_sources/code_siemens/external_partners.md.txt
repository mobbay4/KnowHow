# External Partners

## Start a collaboration with the partner

Follow the [Help on code.siemens documentation][code_siemens_help_partners].

If the partners must access code.siemens via the internet see [this faq][code_siemens_help_internet].

## End the collaboration with the partner

A GID can not be deleted, see [IAM Help on GID][IAM_delete_GID].

If a collaboration comes to end, set the date in IAM to the date the collaboration ends.
See the [IAM Help for extending the end date][IAM_extend_end_date]

```{hint}
Refer to [following help page][IAM_how_to_access_IAM] to access IAM.
```

After the end date is reached, the Sponsor receives a myIT Ticket with the offboarding process of the business partner.
The sponsor must configure and activate the ticket.
