# General Hints

## Embed Excel and CSV files in Sphinx

There are some possebilities to embed table data into a Sphinx documentation.

### .XLS-files

To embed a `.xls` file in your Sphinx documentation you can use the Sphinx extension [sphinxcontrib-exceltable]. Which provides an `.rst` directive that can be used like this to embed your excel file into your documentation:

```rst
My document
===========
The contents of the setup script:

.. exceltable:: Table caption
   :file: path/to/document.xls
   :header: 1
   :selection: A1:B3
   :sheet: SheetName
```

When you go onto the [sphinxcontrib-exceltable] website you can find many possibilties to configure how the table should be embeded.

Further links:

- [Short Article]

### .CSV-files

To embed a `.csv` file in your Sphinx documentation you can use the Sphinx-native [csv-table] directive like this:

```rst
.. csv-table:: The contents of my xlsx sheet exported to mytable.csv
   :widths: 15 40 20
   :header: "Header 1", "Header 2", "Header 3"
   :file: test_datei.csv
```

options:

- `width` - gives the % each column has. The sum must be 100%. The width option is optional.
- `header-rows` - how manny rows of the table are shown as header
