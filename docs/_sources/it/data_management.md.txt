# Data Management

## Ordner vergleichen

[Dieser Artikel][compare_folders] mit welchen Programmen Ordner verglichen werden können. **Der Abschnitt zu Robocopy ist sehr hilfreich**. Aus diesem Abschnitt sind die Befehle im Kapitel Robocopy.

2022-03: WinMwerge kann nicht so eingestellt werden, dass es nur die Dateinamen vergleicht. Siehe [dieses Issue][WinMerge_folder_diff].

### Robocopy

Mit folgendem Befehl kann man zwei Ordner vergleichen:

```PowerShell
robocopy "Pfad/Ordner1" "Pfad/Ordner2" /L /MIR /LOG:E:\Diff.txt
```

```{note}
Die allgemeine Bedienung von Robocopy ist auf der [Robocopy Seite][Robocopy] erklärt. Dort gibt es eine Liste mit häufig [benötigten Parametern][Robocopy_parameter].
```

## Inhalt von Dateien vergleichen

Wenn der Inhalt von Dateien verglichen werden soll: WinMerge.
