# Shells

Beschreibung von Shells nach [Wikipedia][shells_wikipedia]:
In der Informatik wird als Shell die Software bezeichnet, mittels derer ein Benutzer mit einem Betriebssystem interagiert[1] - eine Mensch-Maschine-Schnittstelle. Während Kernel den Kern eines Betriebssystems bezeichnet, ist die Shell (englisch für „Schale", „Hülle" oder „Außenhaut") anschaulich dessen Außenschicht und damit Schnittstelle (englisch Interface) zum Benutzer.

## Aufteilung dieses Abschnitts

Allgemein gültiges Wissen zu Shells wird in diesem Dokument gesammelt.

Wissen zu spezifischen Shells, wie PowerShell, wird in Unterkapiteln gesammelt.

## Befehle der Shell Logik

- `|` Die Ausgabe des einen Befehls wird an den nächsten übergeben
- `>` Die Ausgabe wird in eine Datei geschrieben. Die Datei wird neu erstellt, auch wenn sie schon exzisiert.
- `>>` Die Ausgabe wird in eine Datei geschrieben. Wenn die Datei schon exzisiert wird der Inhalt angehängt.

[shells_wikipedia]: https://de.wikipedia.org/wiki/Shell_(Betriebssystem)
