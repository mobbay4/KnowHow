Markdown file outside docs folder in Markdown#

To include a markdown / rst file outside the docs folder to a markdown / rst file inside the docs folder use the myST Extension: MySt: Inlclude markdown outside docs

Warning

Links werden vom “importierten File” in das “importierenden File” eingelesen und dann erst interpretiert. Dadurch werden die Links relativ zum “importierenden File” angegeben. Die Dokumentation beachten: MySt: Inlclude markdown outside docs.

Hier noch der Originaltext: However, including a file will not usually resolve local links correctly, like ![](my-image.png), since it treats the text as if it originated from the “including file”.

2021.12: The dependency “The parser option requires docutils>=0.17” was fullfiled with “standard” installation.

```{include} ../README.md
```

include rst files outside docs folder in Markdown#

The following Markdown “Code” is needed:

```{toctree}
:caption: Contribute
:hidden:

../../CONTRIBUTING.rst

```
